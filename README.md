## Мое портфолио 

Название проекта: Console Bank App (Первый проект)

Кратко:
Это консольное приложение, которое имитирует простейшую работу банка. Я написал его, когда только начал изучать программирование, чтобы понять, как соединяются базовые конструкции: переменные, условия, циклы и функции.

Что умеет приложение:

Создать новый «счёт» (просто в памяти).

Положить деньги на счёт.

Снять деньги (с проверкой - нельзя уйти в минус).

Показать текущий баланс.

Выход из программы.

Технологии / стек:
Язык: C#
Среда: Visual Studio / VS Code
